# x1scroll MEV Smoothing Pool

**Cross-epoch MEV income smoothing for X1 validators.**

Validators pool MEV earnings. Every epoch the pool distributes equally to all members — smoothing lumpy MEV income into predictable yield.

## How It Works

1. Pay 5 XNT to join the pool
2. Contribute MEV earnings each epoch
3. Pool distributes equally to all active members once per epoch
4. x1scroll takes 10% operator fee (50% treasury / 50% burned 🔥)

## Fee Structure (immutable)

| Fee | Amount | Split |
|-----|--------|-------|
| Registration | 5 XNT | → treasury |
| Operator fee | 10% per distribution | 50% treasury / 50% burned |
